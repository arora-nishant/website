# Files Directory

Place your resume PDF and other downloadable files here.

## Resume

Add your resume PDF file as `resume.pdf` in this directory.

The resume will be:
- Displayed in an embedded viewer on the Resume page
- Available for download via the download button

**To add your resume:**
1. Export/save your resume as a PDF
2. Name it `resume.pdf`
3. Place it in this directory (`files/resume.pdf`)

**Tips for your resume PDF:**
- Keep file size reasonable (under 2MB)
- Ensure it's readable and well-formatted
- Test that it displays correctly in a browser
- Make sure all fonts are embedded

## Other Files

You can also store other downloadable files here:
- Portfolio PDFs
- Case studies
- White papers
- Presentations

Reference them in your HTML:
```html
<a href="files/your-document.pdf" download>Download Document</a>
```

Or for inline viewing:
```html
<iframe src="files/your-document.pdf"></iframe>
```
